# 微剧场 - 影视短剧发布平台

一个现代化的影视短剧展示和分享平台，支持管理后台和一键部署到 Vercel。

## 功能特性

- 🎬 **精美影院风格设计，沉浸式体验
- 📱 **完全响应式，支持桌面和移动端
- 🔗 **支持夸克/百度/阿里云盘等多平台资源链接
- 🔄 **管理后台，支持添加/编辑/删除剧集
- 📤 **单个剧集的分享功能
- 🌟 **首页精选推荐
- 🎯 **标签系统（独家、热播、新作等）
- ⚡ **Next.js + Vercel 一键部署

## 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 部署到 Vercel

1. 将代码推送到 GitHub 仓库
2. 在 Vercel 导入项目
3. 部署后自动运行

或者点击下方按钮一键部署：

[![Deploy with Vercel](https://vercel.com/button)

## 使用说明

### 访问首页

- 打开首页即可看到所有剧集列表，包含海报、剧名、简介和网盘链接
- 点击剧集卡片有悬停动画效果
- 点击分享按钮可直接分享或复制链接

### 管理后台

访问 `/admin` 进入管理后台：

- **添加剧集**：点击「添加新剧集」按钮，填写剧集信息
- **编辑剧集**：点击「编辑」按钮修改现有剧集
- **删除剧集**：点击「删除」按钮删除剧集
- **设为推荐**：编辑时勾选「设为首页推荐」可将剧集设为 Hero 区展示

### 剧集信息字段

| 字段 | 说明 | 必填 |
|------|------|
| 剧名 | 剧集名称 | ✅ |
| 简介 | 剧情简介 | ✅ |
| 海报 | 海报图片链接 | ✅ |
| 集数 | 总集数 | |
| 年份 | 年份 | |
| 类型 | 剧集类型（如：都市情感） | |
| 标签 | 特殊标签（如：独家） | |
| 资源链接 | 网盘链接 | ✅ |
| 网盘类型 | 夸克/百度/阿里 | |
| 画质 | HD/1080P/4K | |
